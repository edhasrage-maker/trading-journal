# Phone notes — inbox

Append-only inbox for quick thoughts captured from phone about the
take-public plan (`TAKE_PUBLIC_PLAN.md`). Anything dropped here gets
integrated into the planning doc on the next desktop Claude Code session,
then this file gets cleared.

## How to use this from phone

**Easiest path (no AI on phone):** Open this file via GitHub mobile app or
github.dev (append `.dev` to the repo URL in browser), scroll to the bottom,
add a dated entry, commit. Done.

**With AI conversation on phone:** Open Claude.ai mobile app, start a fresh
chat. Paste in the relevant section of `TAKE_PUBLIC_PLAN.md` as context
(or the whole thing — it's not that long). Have the conversation. When done,
ask Claude.ai: *"Summarize the decisions / questions / changes from this
conversation as a markdown block I can paste into a planning notes file."*
Paste that block here.

Either way, on the next desktop session: "read `docs/PHONE_NOTES.md`,
integrate into `docs/TAKE_PUBLIC_PLAN.md`, and clear the inbox."
